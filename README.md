# AI Chatbot Assistant 🤖

An interactive **AI-powered chatbot web application** built using **Python and Flask**.
The chatbot assists users by answering questions related to **bus transport services**, including bus timings, routes, ticket prices, and bus pass applications.

This project demonstrates a **retrieval-based conversational AI system** that analyzes user queries and returns relevant predefined responses through a simple web interface.

---

## 🚀 Features

* Interactive chatbot interface
* Natural language style user queries
* Bus timing and schedule information
* Bus routes guidance
* Ticket price details
* Student bus pass assistance
* Bus pass application guidance
* Quick suggestion buttons for common questions
* Typing animation for realistic chatbot responses
* Responsive and clean UI design

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/YOURUSERNAME/ai-chatbot-assistant.git
```

### 2. Navigate to the project folder

```
cd ai-chatbot-assistant
```

### 3. Install dependencies

```
pip install flask
```

### 4. Run the application

```
python app.py
```

### 5. Open the application in your browser

```
http://127.0.0.1:5000
```
